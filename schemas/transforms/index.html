Spatial Transform Types Vocabulary

Namespace: https://example.org/spatial-transforms/

Preferred Prefix: st:

Version: 1.0.0

License: CC BY 4.0

Download: Turtle JSON-LD Context Terms List

This vocabulary defines types of spatial coordinate transformations for use in scientific datasets, particularly for image registration and coordinate system mapping.

Transform Types

Identity Transform

identity

Maps input coordinates to output coordinates without modification.

The identity transformation that leaves coordinates unchanged.

Translation Transform

translation

Translates points by a fixed vector.

A transformation that shifts all points by a constant vector.

Parameters:

translation number[] required

Translation vector with length N

Scale Transform

scale

Scales coordinates by factors along each axis.

A transformation that scales coordinates by specified factors.

Parameters:

scale number[] required

Scale factors with length N (non-zero values)

Rotation Transform

rotation

Rotates coordinates using an orthonormal matrix.

A transformation that rotates coordinates in N-dimensional space.

Parameters:

matrix number[][] required

NxN rotation matrix with determinant 1

Affine Transform

affine

General affine transformation from N to M dimensions.

A transformation that preserves lines and parallelism.

Parameters:

matrix number[][] required

Mx(N+1) affine transformation matrix

Map Axis Transform

mapAxis

Permutes axes by mapping axis indices.

A transformation that reorders dimensions.

Parameters:

mapping object required

Object mapping output axis names to input axis names

Sequence Transform

sequence

Composition of multiple transformations applied in order.

A transformation that applies multiple transforms sequentially.

Parameters:

transformations transform[] required

Ordered list of transformations to apply

Displacement Field Transform

displacements

Transform using a displacement vector field.

A transformation defined by displacement vectors at each point.

Parameters:

field string required

Reference to displacement field array

interpolation string optional

Interpolation method: linear (default), nearest, or cubic

Coordinate Field Transform

coordinates

Transform using absolute coordinate mappings.

A transformation defined by absolute coordinate values.

Parameters:

field string required

Reference to coordinate field array

interpolation string optional

Interpolation method: linear (default), nearest, or cubic

Inverse Transform

inverseOf

Specifies the inverse of another transformation.

A transformation that reverses another transformation.

Parameters:

transform transform required

The transformation to invert

Bijection Transform

bijection

Explicitly defined forward and inverse transformations.

A transformation with explicit forward and inverse mappings.

Parameters:

forward transform required

Forward transformation

inverse transform required

Inverse transformation

By Dimension Transform

byDimension

Applies different transformations to subsets of dimensions.

A transformation that operates on dimension subsets independently.

Parameters:

transformations dimensionTransform[] required

List of transformations with dimension specifications
